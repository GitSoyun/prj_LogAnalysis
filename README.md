# Mini Project: Log Analysis Program
Java, 검색정보 log파일을 정량적으로 분석해주는 프로그램

<sub>with 김현규, 서현익, 신지영, 최은혜</sub>

---

## 담당 업무
* 정소윤
  - 전체 Swing 디자인 구현 
  - List, Set, Map을 이용해 초기 설계
  - 기능 구현

---

## 의의
* Java의 자료구조를 이용한 문제해결 능력 향상
* 사용자(사이트 관리자)에게 원하는 정보 제공
* 정량적 분석(어떤 대상이 얼마나 존재하는지)에 대한 연습
